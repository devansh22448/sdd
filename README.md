## ✨ Overview

**Smart DevOps Dashboard** is an enterprise-grade CI/CD automation tool. It listens to your GitHub repositories, automatically triggers build processes in an isolated environment, and streams real-time terminal logs directly to a beautiful, dark-themed React dashboard.

## 🚀 Key Features

* **🤖 Auto-Deployments via Webhooks:** Instantly detects `git push` events using GitHub Webhooks and Smee.io.
* **⚡ Live Terminal Streaming:** Watch your code clone and build in real-time with WebSockets (Socket.io).
* **⚙️ Custom Build Engine:** Executes real bash/terminal commands (like `npm install`) using Node.js `child_process`.
* **📊 Visual Analytics:** Beautiful Recharts integration to monitor deployment success rates and history.
* **✅ GitHub Status Sync:** Sends live Pending/Success/Failure checks directly back to your GitHub commits.
* **🕰️ Time Machine Logs:** MongoDB persistence ensures you never lose past deployment logs.
